# UrbanaFix

## 🇧🇷 Descrição (Português)

UrbanaFix é uma plataforma web criada para conectar pessoas que precisam resolver problemas urbanos com quem pode ajudar a solucioná-los. A ideia do projeto é permitir que usuários publiquem ocorrências da cidade, como buracos, iluminação quebrada, lixo acumulado ou outros problemas urbanos, facilitando a comunicação e a colaboração entre a comunidade.

O sistema possui funcionalidades como cadastro de usuários, login, publicação de posts, feed de ocorrências e interação entre usuários. O projeto foi desenvolvido utilizando **PHP, HTML, CSS e JavaScript**, com foco em simplicidade, organização do código e experiência do usuário.

## 🇺🇸 Description (English)

UrbanaFix is a web platform created to connect people who want to report urban issues with those who can help solve them. The goal of the project is to allow users to publish city-related problems such as potholes, broken street lights, garbage accumulation, or other urban issues, making communication and community collaboration easier.

The system includes features such as user registration, login, post creation, an issue feed, and user interaction. The project was developed using **PHP, HTML, CSS, and JavaScript**, focusing on simplicity, clean code organization, and user experience.
